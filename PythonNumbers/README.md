            <h2>Python Numbers</h2>
            <p>There are three numeric types in Python:</p>
            <ul>
                <li><code>int</code> </li>
                <li><code>float</code> </li>
                <li><code>complex</code> </li>
                <p>Example.</p>
                <ul>
                    <li>
                        <code>
                            x = 1    # int
                            y = 2.8  # float
                            z = 1j   # complex
                        </code>
                    </li>
                </ul>
                <p>To verify the type of any object in Python, use the type() function:</p>
                <ul>
                    <li>
                        <code>
                            print(type(x))
                            print(type(y))
                            print(type(z))
                        </code>
                    </li>
                </ul>
            </ul>
